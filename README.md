### Install

#### Clone this repo

```
git clone https://github.com/alexdevero/zentime-meditation-web-app.git
```

#### Install dependencies

```
npm install
```
or
```
yarn
```

### Usage

#### Run the project (on localhost:1337)

```
npm run start
```
or
```
yarn run start
```

#### Preview the minified production bundle (on localhost:1234)

```
npm run preview
```
or
```
yarn run preview
```

#### Build project

```
npm run build
```
or
```
yarn run build
```
